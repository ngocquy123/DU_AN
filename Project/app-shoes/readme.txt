Để khởi chạy dự án 
Bước 1: Bạn khởi chạy xamp hoặc 1 phần mềm chạy server ảo nào đó.
Bước 2: Bạn vào thư mục Backend - Database
Lấy file intern-php.SQL database và khởi chạy server ảo sau đó vào MYSQLI - PHPADMIN
bạn đã import xong.
Bước 3: Bạn bạn xamp và chọn đến file index.html và chạy nó.
Bước 4: Bạn vui lòng kiểm tra đường dẫn URL của fetch có giống không.
vì ở máy tôi thì đường dẫn thư mục bị khác, không giống bạn nên máy bạn kiểm tra lại xem thử.
Bước 5: Sau khi khởi chạy xong bạn cứ tha hồ test nha, xin cảm ơn.
